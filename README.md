## 📌 درباره Go

Go یک زبان برنامه‌نویسی سریع، ساده و قدرتمند است که توسط Google ساخته شده است.  
این زبان برای ساخت:

- Backend
- API
- Microservices
- CLI Tools
- Cloud Applications
- Concurrent Programs

بسیار محبوب و کاربردی است.
